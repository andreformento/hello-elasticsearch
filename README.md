# hello-elasticsearch

- start: `make start`
- user `elastic`
- password: `cat .env | grep ES_LOCAL_PASSWORD`
- Elasticsearch will be running at http://localhost:9200
- Kibana will be running at http://localhost:5601
- Jupyter Notebook will be running at http://localhost:8888
